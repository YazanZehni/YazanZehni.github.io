# TrustHLT application portfolio

This is a static portfolio website tailored for the student assistant position in website creation and prototype design.

## Contents
- `index.html` – single-page portfolio
- `styles.css` – styling
- `script.js` – bilingual text switch + small interactive prototype demo
- `documents/` – CV and transcript PDFs copied from the source files
- `assets/favicon.svg` – small local favicon

## Deploy on GitHub Pages
1. Create a repository, for example `username.github.io` or `portfolio`.
2. Upload all files from this folder.
3. In GitHub: **Settings → Pages → Deploy from a branch**.
4. Select the `main` branch and the root folder.
5. Wait a few minutes until the site is published.

## Notes
- No framework or build process is needed.
- Everything is self-contained and works as a simple static website.
- The portfolio is intentionally honest: it only uses details visible in the provided CV and transcript.
